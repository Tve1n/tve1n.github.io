Alien Invasion

Небольшая аркадная игра

Описание проекта

Игра, в которой пользователь должен управлять кораблём, чтобы уничтожить всех пришельцев на экране и перейти на следующий уровень.

В данном проекте отработал навыки
  • Углубил знания по работе с классами
  • Работа с библикотекой Pygame
  • Научился отображать информацию в текстовом и графическом виде
  • Использовал вложенные циклы для создания сетки элементов
  • Отслеживал статистику в игре и использовал флаги для определения окончания игры.

Использованные технологии

Python Pygame Git
